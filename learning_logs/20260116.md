## 태그 속성 사용하기

### 태그의 속성(=특성)
- html의 태그의 속성을 영어원문상 python과 동일하게 attribute를 사용함. 그런데 mbn문서에서 전에 말했던 거서럼 특성이라고 번역하는 중인데, 현재 국내의 대부분의 강의에서 속성이라고 표기하고 있고 현직에서도 그러는 중이라서 속성이란 표현을 중심으로 강의함.

- 예시
  1. `<html lang = "ko">`
  2. `<img src="" alt=">`
이상의 예시에서 img태그는 그림을 넣어주는 태그인데, 원본 파일명이 src속성에 들어가고 만약 그림파일이 없을경우 대신 표시할 alt(attribute) 속성이 존재함. 근데 이img 태그내의 sec, alt속성은 p태그나 h1태그에는 존재하지않음. 즉 _태그의 동작을 설정 및 제어하는것_ 이 **태그의 속성(특성)** 이며, 각 태그마다 설정 가능한 속성들이 있음.

`![alt text](image.png)`
`<img src="image.png" alt="안나올땐 얘가 출력됨.>`

## `<form>` 태그 사용하기
- 사용자로부터 입력을 받기위한 양식을 작성하는 태그를 통틀어서 지칭하는 용어.
### `<form>` 태그의 속성
```html
<body bgcolor="5f5f5f">
  <h1>01_formtag.html</h1>
  <form action="./formtag.html" method="get">
    <label for="name">이름</label>
    <input id="name" name="my-name" type="text">
    <br><br>
    <label for="age">나이</label>
    <input id="age" name="my-age" type="number">
    <br><br>
    <button type="submit">제출</button>
    <button type="reset">초기화</button>
  </form>
</body>
```
- 이상의 예시 코드에서 사용되는 속성들
  1. `<form>` 태그 
    - **action** 
    - **method**
      - 현재 수준에서 경험해보지는 못했지만 우리가 이름과 나이를 입력했을 때 각 정보를 가져가서 보여주는 결과 페이지에 해당. 현재 수준에는 _form태그 내에 `action/method` 속성이 있다_ 는 것만 알아도 됨.

  2. `<input>` 태그
      - 이건 아예 챕터를 따로 빼서 설명할 예정
      - `name` : HTML상에서는 특별한 의미를 갖는게 아니라 서버에서 각 항목을 받을 때 _무엇에 대한 값인지 구분 할수 있도록 구분_ 하는 용도에 해당. _DB와의 연결성과 관련._

  3. `<label>` 태그 : `input` 태그와 유기적으로 연결됨.     `input`태그에 부여한 `id 속성 값`을 `label` 태그에서의 `for 속성`에서 불러오기 때문.

  4. `<button>` 태그
      - `type` : `submit / reset`으로 사전에 정의가 되어있음. submit의 경우 입력한 결과값을 최종 서버나 특정 페이지 주소로 보내는, 즉 제출하는 용도. reset은 입력한 내용을 모두 초기화.
